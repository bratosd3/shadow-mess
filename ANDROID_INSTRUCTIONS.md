# Инструкция: Создание Android приложения (APK) для Shadow Message (Discord Clone)

Чтобы превратить ваш веб-интерфейс в полноценное Android-приложение, мы используем **Capacitor**. Это современный инструмент, который позволяет обернуть ваш HTML/JS/CSS сайт в нативное приложение.

## Шаг 1: Подготовка
Убедитесь, что у вас установлены:
1. **Node.js** (уже есть, раз проект работает).
2. **Android Studio** (для компиляции APK).

## Шаг 2: Установка Capacitor в проект

Откройте терминал в папке проекта `d:\программы\auto start\mess` и выполните команды:

```bash
# Инициализация npm (если еще не сделано)
npm init -y

# Установка ядра Capacitor
npm install @capacitor/core @capacitor/cli @capacitor/android

# Инициализация Capacitor (Укажите имя приложения, например "Shadow Discord", и ID пакета "com.shadow.discord")
npx cap init
```

## Шаг 3: Настройка путей

Capacitor берет "статические" файлы (HTML, CSS, JS) и упаковывает их в приложение.
В вашем проекте статические файлы лежат в папке `static`.

1. Откройте файл `capacitor.config.json` (он появится после шага 2).
2. Найдите опцию `"webDir"` и измените её на `"static"`.
   
   Пример `capacitor.config.json`:
   ```json
   {
     "appId": "com.shadow.discord",
     "appName": "Shadow Discord",
     "webDir": "static",
     "bundledWebRuntime": false
   }
   ```

## Шаг 4: Подключение к серверу (Важно!)

Ваше приложение работает в браузере и подключается к `server.js`.
В телефоне нет `localhost`. Вам нужно указать IP-адрес вашего компьютера или адрес хостинга.

1. Откройте `static/js/app.js` (или `calls.js`, где у вас инициализация сокета).
2. Найдите строку подключения, обычно это `io()` или `io('http://localhost:3000')`.
3. Измените на IP вашего компьютера в локальной сети, например: `io('http://192.168.1.5:3000')`.
   *Если вы выложили сервер в интернет (на VPS/Heroku), укажите реальный домен.*

## Шаг 5: Создание Android проекта

Выполните в терминале:

```bash
# Добавляем платформу Android
npx cap add android

# Копируем ваши web-файлы в Android проект
npx cap sync
```

После этой команды в папке проекта появится папка `android`. Это и есть ваш проект для создания APK.

## Шаг 6: Сборка APK

1. Откройте папку `android` в **Android Studio**.
   Или выполните команду:
   ```bash
   npx cap open android
   ```
2. Дождитесь конца синхронизации Gradle (внизу справа будет полоска загрузки).
3. В верхнем меню выберите **Build -> Build Bundle(s) / APK(s) -> Build APK(s)**.
4. После завершения Android Studio покажет уведомление "APK(s) generated successfully". Нажмите **locate**, чтобы найти файл `.apk`.

Теперь этот файл можно перекинуть на телефон и установить!

---

### Полезные команды (для обновления)

Если вы изменили HTML или CSS и хотите обновить приложение на телефоне:

1. Сохраните файлы.
2. Выполните:
   ```bash
   npx cap sync
   ```
3. Снова нажмите Run (Play) в Android Studio или соберите новый APK.
